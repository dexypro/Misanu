Zadatak 399: (File01)

Definisati pokretačku klasu Main i u njoj: 
- kreirati objekat klase File i pozvati metode exists(), isDirectory(), canRead(), lastModified().

Pokretačka klasa:

package f1;

import java.io.File;

public class File01 {

  public static void main(String[] args) {
    File f = new File("C:\\Users\\bajov\\Desktop\\test.java");

    System.out.println("Fajl (" + f.getName() + (f.isFile() ? ") je fajl." : ") nije fajl."));
    System.out.println("Apsolutna putanja: " + f.getAbsolutePath());
    System.out.println(f.exists() ? "Fajl postoji." : "Fajl ne postoji.");
    System.out.println(f.isDirectory() ? "Direktorijum: " : "Fajl");
    System.out.println(f.canRead() ? "Fajl može biti procitan." : "Fajl ne može biti pročitan.");
    System.out.println(
      f.canWrite() ? "U fajlu se može vršiti upisivanje." : "Nije omogućeno upisivanje u ovaj fajl.");
    System.out.println("Veličina: " + f.length());
    System.out.println("Poslednja promena fajla: " + f.lastModified());

  }

}

Fajl postoji.

Fajl može biti procitan.
U fajlu se može vršiti upisivanje.

Poslednja promena fajla: 1676039022161

Zadatak 400: (File02)

Definisati pokretačku klasu Main i u njoj: 
- kreirati objekat dis klase DataInputStream.
- kreirati objekat fos klase FileOutputStream.
- omogućuti unos texta u fajl

Pokretačka klasa:

package f2;

import java.io.DataInputStream;
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.IOException;

public class Main {

  public static void main(String[] args) throws IOException {

    /* Osnovne UI operacije
    Za UI operacije Java koristi tokove(Stream)
    Strimovi obezbeđuju način za rad sa podacima koji se primaju i šalju */

      /* Klasa koja omogućuje da se poveže tastatura sa stream-om */
    DataInputStream dis = new DataInputStream(System.in);

    /* Povezivanje fajla sa klasom FileOutputStream */
    FileOutputStream fos = new FileOutputStream("rezultat.txt");

    System.out.println("Unesite string (prekinite unosom znaka $): ");

    // read ()
    // write()

    char ch;

    while ((ch = (char) dis.read()) != '$') {
        fos.write(ch);
    }
    fos.close();

  }

}

Unesite string (prekinite unosom znaka $): 
Ovo je text koji se upisuje u fajl!$
