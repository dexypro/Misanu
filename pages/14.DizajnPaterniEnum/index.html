Kreirati dizajn patern: Adapter

Interfejs:

package z1;

public interface Curka {

  public void curlice();

  public void leti();
}

Interfejs:

package z1;

public interface Patka {

  public void kvace();

  public void leti();
}

Klasa:

package z1;

/* 1. Prvo se implementira interfejs tipa prema kojem se vrsi prilagodjavanje
  * to je interfejs koji klijent ocekuje */
public class CurkaAdapter implements Patka {

  Curka curka;

  /*2. Dobiti referencu na objekat koji prilagodjavamo*/

  public CurkaAdapter(Curka curka) {
    this.curka = curka;
  }

   /*3. Implementacija metoda u interfejsu */

  /* Prevodjenje kvace()* u metodu curlice()  izmedju klasa */
  public void kvace() {
    ka.curlice();
  }

  public void leti() {
    for (int i = 0; i < 5; i++) {
        curka.leti();
    }
  }

}

package z1;

public class Main {

  static void testPatka(Patka patka) {
    patka.kvace();
    patka.leti();
  }

  public static void main(String[] args) {
    DivljaPatka patka = new DivljaPatka();
    Curka curka = new DivljaCurka();

    /* Umotavamo Curku u CurkaAdapter, zahvaljujuci kome ce izgledati kao Patka */
    Patka curkaAdapter = new CurkaAdapter(curka);

    System.out.println("Curka: ");
    curka.curlice();
    curka.leti();

    System.out.println("Patka: ");
    testPatka(patka);

     /* Pokusavamo da poturimo Curku kao Patku */
    System.out.println("AdapterCurka");
    testPatka(curkaAdapter); /* ne zna da je dobila Curku prerusenu u Patku */
  }

}

Pokretačka klasa:

package e4;

public class Main {

  
// ordinal() - dobija redni broj tj. polozaj date konstante na listi 
konstanti u nabrajanju. Redni brojevi pocinju od 0.

// compareTo() - poredi redne brojeve dveju konstanti iz istog nabrajanja. 
Ako je redni broj pozivajuce konstante manji od rednog broja konstante
// metoda compareTo() vraca negativan broj. Ukoliko su redni brojevi jednaki 
metoda comareTo() vraca 0. Ukoliko pozivajuca konstanta ima redni broj veci od 
rednog broja konstante metoda compareTo() vraca pozitivan broj

// equals() - poredi se jednakost konstante iz nabrajanja sa svakim drugim objektom 

  public static void main(String[] args) {

    System.out.println("Boja - redni broj");

    for (Boje bb : Boje.values()) {
        System.out.println(bb + " " + bb.ordinal());
    }

    System.out.println();

    Boje b1, b2, b3, b4;

    b1 = Boje.Crvena;
    b2 = Boje.Zelena;
    b3 = Boje.Plava;
    b4 = Boje.Crvena;

    if (b1.compareTo(b2) < 0) {
        System.out.println(b1 + " dolazi pre " + b2);
    }
    if (b1.compareTo(b2) > 0) {
        System.out.println(b2 + " dolazi pre " + b1);
    }
    if (b1.compareTo(b1) == 0) {
        System.out.println(b1 + " jednako " + b1);
    }

    if (b1.equals(b2)) {
        System.out.println("Greska");
    }
    if (b1.equals(b3)) {
        System.out.println(b1 + " jednako " + b3);

        System.out.println(b1 + " jednako " + b3);

  }

Zadatak 398: (Enum05)

Definisati javni enum Odgovori i u njoj:
- konstante "DA, NE, MOZDA, KASNIJE, USKORO, NIKAD".

Definisati klasu EnumPrikaz i u njoj:
- Switch / Case  u kojoj se se za svaku konstantu iz navedenog enuma ispisuje njen naziv.

Definisati klasu Pitanje i u njoj:
- metodu odg enuma Odgovori kojom se proverava koja se konstanta ispisuje na osnovu inicijalizovane vrednosti od 15.

Definisati pokretačku klasu Main i u njoj: 
- kreirati objekat klase Pitanje i pozvati metodu odg tri puta.

Klasa:

package e5;

public enum Odgovori {

    DA, NE, MOZDA, KASNIJE, USKORO, NIKAD

}

Klasa:

package e5;

import java.util.Random;

public class Pitanje {

  Random rand = new Random();

  Odgovori odg() {

    int vrednost = (int) (rand.nextDouble() * 100);

    if (vrednost < 15) {
        return Odgovori.MOZDA;
    } else if (vrednost < 30) {
        return Odgovori.DA;
    } else if (vrednost < 60) {
        return Odgovori.NE;
    } else if (vrednost < 75) {
        return Odgovori.KASNIJE;
    } else if (vrednost < 95) {
        return Odgovori.USKORO;
    } else
        return Odgovori.NIKAD;
  }
}
