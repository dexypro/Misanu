Zadatak 350: (ForEachPetlja)

Napisati program i simulirati Unapređenu For petlju.

Pokretačka klasa:

package unapredjenaForPetlja;

import java.util.Scanner;

public class ForEachPetlja {

    public static void main(String[] args) {

  Scanner ulaz = new Scanner(System.in);

  /* Prvi nacin kreiranja niza */
  int[] brojevi = new int[5];

  /* Drugi nacin kreiranja niza */
  int niz[] = { 2, 2, 2, 2, 2, 2 };

  /* Unos podataka za niz brojevi*/
  for (int i = 0; i < brojevi.length; i++) {
      System.out.print("brojevi[" + i + "] = ");
      brojevi[i] = ulaz.nextInt();

      if ((i + 1) == brojevi.length) {
        System.out.println("Niz je napunjen!");
      }
  }

  /* Unapredjena FOR petlja, tkz ForEach */
  System.out.println("\nIzgled niza 'brojevi':");
  for (int i : brojevi) {
      System.out.print(i + ", ");
  }

  System.out.println("\nIzgled niza 'niz':");
  for (int j : niz) {
      System.out.print(j + ", ");
  }

  ulaz.close();
    }

}

Izlaz:

brojevi[0] = 1
brojevi[1] = 2
brojevi[2] = 3
brojevi[3] = 4
brojevi[4] = 5
Niz je napunjen!

Izgled niza 'brojevi':
1, 2, 3, 4, 5, 
Izgled niza 'niz':
2, 2, 2, 2, 2, 2,

Zadatak 352: (ArrayList02)

Kreirati ArrayList i pozvati i objasniti sledeće metode:
- .size()
- .add()
- .contains()
- .get()
- .indexOf()
- .toArray()
- .remove()
- .clear()

Pokretačka klasa:

package ArrayList2;

import java.util.ArrayList;
import java.util.List;

public class Main {

    public static void main(String[] args) {
  List lista = new ArrayList();
    /* size() - vraca broj elementa smestenih u kolekciji ArrayList */
  System.out.println("Pocetna duzina: " + lista.size());

  /* add() - dodaje element na kraj kolekcije */
  lista.add("Prvi"); // 0
  lista.add("Drugi"); // 1
  lista.add("Treci"); // 2
  lista.add("Cetvrti"); // 3

  /* prolazak kroz kolekciju */
  for (String s : lista) {
      System.out.printf("%s ", s);
  }

  System.out.println("\nTrenutna duzina kolekcije je: " + lista.size());

  /*provera da li se neki zadati element nalazi unutar kolekcije */
  /* contains - vraca true ako kolekcija sadrzi trazeni element, 
    * odnosno vraca false ukoliko kolekcija ne sadrzi trazeni element */

  boolean sadrzi = lista.contains("Treci");
  if (sadrzi) {
      System.out.println("Trazeni element se nalazi u zadatoj kolekciji!");

  } else {
      System.out.println("Trazeni element se ne nalazi u zadatoj kolekciji!");
  }

  /* get() - vraca element sa zadatim indeksom */
  String element = lista.get(2);
  System.out.println("Element sa indeksom 2 u kolekciji je: " + element);

  /* indexOf - vraca indeks prve pojave zadatog elementa kolekcije i 
    * vratice -1 ukoliko trazeni element ne postoji */
  int pozicija = lista.indexOf("Drugi");
  System.out.println("Element Drugi je na poziciji: " + pozicija);

  /* toArray - sluzi za prebacivanje elemenata u niz */
  String[] nizString = new String[lista.size()];
  lista.toArray(nizString);

  /*Prolazak kroz niz*/
  for (String s : nizString) {
      System.out.printf("%s ", s);
  }

  /* remove - brise prvu pojavu zadate vrednosti*/
  System.out.println("\n\nUklanjanja elementa Treci: ");
  lista.remove("Treci");
  for (String s : lista) {
      System.out.printf("%s ", s);
  }

  /*clear - brise sve elemente iz zadate kolekcije*/
  lista.clear();
  System.out.println("\n\nVelicina kolekcije nakon brisanja: " + lista.size());
    }

}

Izlaz:

Trazeni element se nalazi u zadatoj kolekciji!

Element Drugi je na poziciji: 1
Prvi Drugi Treci Cetvrti 

Uklanjanja elementa Treci:

Zadatak 354: (ArrayList04)

Kreirati niz Imena sa elementima "Mika", "Zika", "Ana", "Lana", "Hana", "Una"  i kreirati ArrayList i:
- Unapredjenom for petljom kopirati elemente iz niza u ArrayList i istom petljom ispisati tu listu. 
- napraviti niz imenaZaBrisanje "Mika", "Lana", "Hana".
- koriščenjem metode .remove() proslediti kreirani niz kao parametar.
- kreirati objekat iter klase Iterator i štampati listu primenom metoda .hasNext() i .next().
- u listu dodati element "Una" primenom metode .add() i korišćenjem iteratora pronaći vrednost "Una" i izbrisati iz liste metodom .remove()

Pokretačka klasa:

  package ArrayList4;

  import java.util.ArrayList;
  import java.util.Iterator;
  import java.util.List;
  
  public class Main {
  
      public static void main(String[] args) {
        String imena[] = { "Mika", "Zika", "Ana", "Lana", "Hana", "Una" };
      
        List listaImena = new ArrayList();
      
          /* Dodavanje imena iz niza u kolekciju*/
        for (String s : imena) {
            listaImena.add(s);
        }
      
        /*Ispsiivanje kolekcije*/
        for (String s : listaImena) {
            System.out.printf("%s ", s);
        }
      
        /* Imena za brisanje */
        String imenaZaBrisanje[] = { "Mika", "Lana", "Hana" };
      
        /* Brisanje imena iz kolekcije*/
        for (String s : imenaZaBrisanje) {
            listaImena.remove(s);
        }
      
        /*Ispsiivanje kolekcije*/
        System.out.println("\n\nNovi izgled: ");
        for (String s : listaImena) {
            System.out.printf("%s ", s);
        }
      
        /* Kreiranje iteratora kolekcije i stampanje liste pomocu iteratora*/
        Iterator iter = listaImena.iterator();
        System.out.println("\n\nStampanje elemenata pomocu iteratora!");
        while (iter.hasNext()) {
            System.out.print(iter.next() + " ");
        }
      
        /*Brisanje elemenata liste imena koja se nalaze u listi listaImenaZaBrisanje*/
        List listaImenaZaBrisanje = new ArrayList();
      
        listaImenaZaBrisanje.add("Una");
        iter = listaImena.iterator();
        while (iter.hasNext())
            if (listaImenaZaBrisanje.contains(iter.next()))
          iter.remove();
      
        System.out.println("\n\nLista nakon drugog brisanja: ");
        for (int i = 0; i < listaImena.size(); i++) {
            System.out.print(listaImena.get(i) + " ");
    }
  
      }
  
  }

Kreirati objekat ll klase LinkedList i:
- korišćenjem metode .add() dodati sledeće vrednosti u listu: "A", "B", (na drugoj poziciji) "C".
- ištampati elemente liste.
- obrisati element indexa 0 kao i vrednost "B" koriščenjem metode .remove().
- dodati elemente "D", "E", "F" u listu metodom .add().
- metodom .get() dohvatiti element na poziciji 3.
- metodom .set() postaviti element vrednosti "Z" na poziciji 1.

Pokretačka klasa:

  package LinkedList01;

  import java.util.LinkedList;
  import java.util.List;
  
  public class Main {
    public static void main(String[] args) {
      List ll = new LinkedList();
    
        /* Dodavanje elemanta povezanoj lisit */
      ll.add("A"); // 0
      ll.add("B"); // 1
      ll.add(2, "C"); // 2
    
      System.out.println("Elementi povezane liste su: " + ll);
    
      /*Brisanje elemenata iz povezane liste*/
      ll.remove(0); // brisanje preko indexa
      ll.remove("B"); // brisanje preko vrednosti
    
      System.out.println("Povezana lista nakon brisanja elemenata: " + ll);
    
      ll.add("D");
      ll.add("E");
      ll.add("F");
      /* Unutar interfejsa List, indeksiranje ide od 0. Ne moze se indeksirati stringovnim tipom*/
    
      System.out.println("Elementi povezane liste su: " + ll);
    
      /*Trazenje elemenata u povezanoj listi */
      boolean sadrzi = ll.contains("E");
    
      if (sadrzi) {
          System.out.println("Lista sadrzi element E.");
      } else {
          System.out.println("Lista ne sadrzi element E.");
      }
    
      /*Pristupanje (uzimanje) vrednosti elemenata iz povezane liste*/
      Object element = ll.get(3);
      System.out.println("Zatrazen element je: " + element);
    
      /*Postavljanje vrednosti elemenata u povezanoj listi*/
      ll.set(1, "Z");
      System.out.println("Povezana lista nakon promene: " + ll);
    
    }
}

Izlaz:

Povezana lista nakon brisanja elemenata: [C]

Zadatak 357: (LinkedList03)

Uraditi sledeće:
- kreirati objekat listaImena klase LinkedList i u nju dodati elemente  "1", "2", "3", "4", "5".
- kreirati objekat itr klase Iterator i pozvati metodu .listIterator().
- korišćenjem metode .hasNext() i .next() i ispisati listu od prvog do poslednjeg elementa.
- korišćenjem metode .hasPrevious() i .previous() i ispisati listu od poslednjeg do prvog elementa.

Pokretačka klasa:

package LinkedList03;

import java.util.LinkedList;
import java.util.List;
import java.util.ListIterator;

public class Main {

    public static void main(String[] args) {

      List LList = new LinkedList();

      LList.add("1");
      LList.add("2");
      LList.add("3");
      LList.add("4");
      LList.add("5");

      ListIterator itr = LList.listIterator();

      System.out.println("Kretanje kroz listu od prvod do poslednjeg elementa");
      while (itr.hasNext()) { // 0 .. n
          System.out.print(itr.next() + " ");
      }

      System.out.println("\nKretanje kroz listu od poslednjeg do prvog elementa");

      while (itr.hasPrevious()) { // 0
          System.out.print(itr.previous() + " ");
      }

    }
}

Izlaz:

Kretanje kroz listu od prvod do poslednjeg elementa
1 2 3 4 5 
Kretanje kroz listu od poslednjeg do prvog elementa
5 4 3 2 1
