package Thread01;

public class Thread1 {

  public static void main(String[] args) {

    // pocinje da se izvrsava. To je glavna ili inicijalna nit.

    // metode currentThread() klase Thread.

    System.out.println(Thread.currentThread().getName());
    System.out.println(Thread.currentThread().getPriority());

    // grupa niti je struktura podataka koja upravlja stanjem kolekcije niti kao celinom.
  }

Zadatak 371: (Thread07)

Definisati klasu A koje nasledjuje klasu Thread i u toj klasi:
- neophodnu metodu run u kojoj se  try / catch blokom testira metoda .sleep() klase Thread.

Definisati pokretačku klasu i u njoj:
- kreirati objekat kreirane klase A.
- kreirati niz korpaVoca sa sledećim elementima "Limun", "Mandarina", "Jabuka" i ištampati taj niz uz kašnjenje prilikom ispisivanja od pola sekunde.

Klasa:

package Thread07;

public class A extends Thread {
    public void run() {
      int i = 1;

      while (i <= 5) {
          System.out.println("i: " + i);
          // sleep (vrednostUMilisekundama)

          try {
            Thread.sleep(500);
          } catch (InterruptedException e) {
            e.printStackTrace();
          }
          i++;
      }
    }
    // sleep(milisekunde) - privremeno zaustavlja (suspenduje) izvrsavanje niti
    // argument metode sleep() je interval cekanja u milisekundama 

    // Metoda sleep() u klasi Thread moze da izazove izuzetak InterruptedException
    // Desice se izuzetak ukoliko neka druga nit pozeli da prekine ovu suspendovanu nit.
}

Pokretačka klasa:

package Thread07;

  public static void main(String[] args) throws InterruptedException {
    A a = new A();
  
    a.start();
  
    String[] korpaVoca = { "Limun", "Mandarina", "Jabuka" };
    for (String vocka : korpaVoca) {
        System.out.println(vocka);
        Thread.sleep(500);
    }

  }
