Zadatak 324: (Paket01)

U paketu p1 definisati klasu First koja sadrži:
- podatak a tipa integer sa inicijalizovanom vrednošću od 5,
- metodu view koja ispisuje poruku "Ovo je klasa definisana u paketu P1."
    
U paketu p2 definisati pokretačku klasu ImportPackage i u njoj odrediti sledeće:
- importovati klasu First iz paketa p1 koriscenjem kljucne reci import.
- Kreirati objekat klase First sa nazivom 'f.
- Pokrenuti metodu view objekta f.

Klasa:

package p1;
  /*Paketi služe za grupisanje klase.
* Primera radi, možete napraviti dve klase sa istim nazivom unutar dva 
* različita paketa. Paket je istovremeno mehanizam za imenovanje i za
* upravljanje vidljivošću klasa. 
* 
* Paketi sadrže klase. Klase sadrže polja (fields) i metode (methods).
* Metode sadrže lokalne promenljive. Nema globalnih promenljivih u Javi!
*/

public class First {
    public int a = 5;

    public void view() {
      System.out.println("Ovo je klasa definisana u paketu P1.");
    }
}

Pokretačka klasa:

package p2;

/* Prvi način: prelaskom pokazivač miša preko naziva klase i importovanjem klase */

import p1.First; // Importuje se klasa First iz paketa p1
// import p1.* --> importujete sve klase iz navedenog paketa

public class ImportPackage {

    public static void main(String[] args) {

  First f = new First(); /* Kreiranje objekta klase */
  f.view(); /* Za dati objekat poziv funkcije */
    }
}

Izlaz:

Ovo je klasa definisana u paketu P1.

Zadatak 325: (Paket02)

U paketu p1 definisati klasu Balance koja sadrži:
- polje name tipa String,
- polje bal tipa double,
- parametarizovan konstuktor za unos name i bal.
- metodu show koja ispisuje koliko je račun zadužen.

U paketu p2 definisati pokretačku klasu BalanceImport i u njoj odrediti sledeće:
- importovati sav sadržaj paketa p1.
- kreirati niz current tipa Balance koji sadrži tri inicijalizivana objekta klase Balance.
- ispisati sadržaj niza current pozivanjem metode show za svaki od kreiranih objekta.

Klasa:

package p1;

public class Balance {
      String name;
      double bal;

      public Balance(String n, double b) {
        name = n;
        bal = b;
      }

      public void show() {
        if (bal < 0)
            System.out.println("U minusu: \n");
        System.out.println(name + " = " + bal);
    }
}

Pokretačka klasa:

package p2;

import p1.*;

public class BalanceImport {
    public static void main(String[] args) {

      Balance current[] = new Balance[3];
    
      current[0] = new Balance("Mika", 123.45);
      current[1] = new Balance("Zika", 345.12);
      current[2] = new Balance("Ljubivoje", -12.34);

      for (int i = 0; i < 3; i++) {
          current[i].show();
      }
    }
}

Izlaz:

Mika = 123.45
Zika = 345.12
U minusu: 

Ljubivoje = -12.34

Zadatak 330: (Paket07)

Definisati pokretačku klasu Main i u njoj odrediti sledeće:

- unos za poluprecnik r, sračunati površinu kruga i ispisati rezultat.

Pokretačka klasa:

package p1;

import static java.lang.Math.PI;

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {

      Scanner sc = new Scanner(System.in);

      double r, a;
      System.out.println("Unesite poluprecnik: ");
      r = sc.nextDouble();
      a = PI * r * r;
      System.out.println("Povrsina kruga: " + a);
      sc.close();
    }
}

Izlaz:

Unesite poluprecnik: 
5
Povrsina kruga: 78.53981633974483
